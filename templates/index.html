🎯 Geführte Tour — 3 Minuten
So funktioniert CareBridge
1
Patient & Dashboard
Starten Sie auf dem Dashboard. Sehen Sie den Dokumentationsstatus, Trend-Pfeile und nächste fällige Aktionen auf einen Blick.
→ Dashboard öffnen
2
Intensivkurve ausfüllen
Tragen Sie Vitalwerte stündlich ein. Das System markiert überfällige Felder automatisch rot und gibt Textbausteine für den Pflegebericht vor.
→ Intensivkurve öffnen
3
Übergabe prüfen
Vor Schichtende: Übergabe aufrufen. Die Ampel zeigt ob alles dokumentiert ist. Das SBAR-Protokoll wird automatisch befüllt.
→ Übergabe öffnen
